# Random-Number-Game
🔢 Number Guessing Game (Python)

A simple and fun number guessing game built in Python.
The program generates a random number within a user-defined range, and the player tries to guess it with hints provided after each attempt.

✨ Features
User sets the upper limit for the random number

Random number generation using Python’s random module

Interactive guessing loop with hints (above/below)

Tracks number of attempts until the correct guess

Input validation for better user experience
